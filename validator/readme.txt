Класс валидатора содержит в себе конструктор, куда передается массив правил проверки.
Данные из формы передаются внутри класса через $_POST.

Класс содержит основной метод validate, который содержит  парсер массива с правилами и сами правила,
а так же есть приватный метод setError для записи возникающих ошибок несовпадения с правилами,
которые добавляются в строковой массив или json, чтобы потом при помощи яваскрипта вывести
под нужными полями нужные сообщения.

Если ошибок валидации нет, то метод isValidated возвращает true.
Если ошибки есть, то получаем массив при помощи getErrors и обрабатываем в цикле в основном файле с формой.

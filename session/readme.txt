Для передачи флеш сообщений используются сессии.
Это файлы на сервере, которые создаются автоматически при использовании функции session_start().
Эту функцию нужно вызывать первой строкой в любом файле, который использует сессии.

Для работы с сессиями используется суперглобальный массив $_SESSION[].

Для работы флеш сообщений нужна функция для сохранения данных в "переменную".
Для считывания данных в другом файле нужно вызвать вторую функцию, где аргументов будет только название "переменной".
После вывода данных нужно удалить эту переменную из файла сессии при помощи unset().
При обновлении страницы сообщение не должно появляться повторно.
